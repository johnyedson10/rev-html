# Prompt da Tela de Login

Crie uma tela de login moderna e responsiva usando HTML e Tailwind CSS via CDN.

Direção visual:
- Fundo principal em `slate-900`
- Destaques e botão principal em `blue-600`
- Textos em `white`
- Textos secundários e labels em `slate-400`

Requisitos:
- Layout centralizado vertical e horizontalmente
- Card com boa hierarquia visual e bordas suaves
- Título, subtítulo, campos de e-mail e senha
- Link para recuperação de senha
- Botão de entrar bem destacado
- Área opcional para "lembrar-me"
- Responsivo para mobile, tablet e desktop
- Visual limpo, moderno e com contraste forte

Objetivo:
Criar uma experiência de login elegante, clara e fácil de usar em qualquer tamanho de tela.
