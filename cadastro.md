# Prompt da Tela de Cadastro

Crie uma tela de cadastro moderna e responsiva usando HTML e Tailwind CSS via CDN.

Direcao visual:
- Fundo principal em `slate-900`
- Destaques e botao principal em `blue-600`
- Textos em `white`
- Textos secundarios e labels em `slate-400`

Requisitos:
- Layout centralizado com card elegante e contraste forte
- Campos obrigatorios para `nome`, `telefone`, `email`, `senha` e `confirmar senha`
- Validacao visual dos campos com mensagens claras
- Confirmacao de senha obrigatoria e comparacao com a senha principal
- Campo de telefone com mascara simples durante a digitacao
- Botao de cadastro destacado
- Link para voltar para a tela de login
- Responsivo para mobile, tablet e desktop

Objetivo:
Criar uma experiencia de cadastro clara, moderna e segura, mantendo a mesma identidade visual da tela de login.
